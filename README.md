# Karpunets ERC20 token

```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
npx hardhat run --network {network} scripts/deploy.js
npx hardhat verify --contract "contracts/KarpunetsToken.sol:KarpunetsToken" --network {network} {address}
npx hardhat help
```

.env file
```dotenv
ALCHEMY_KEY=
ETHERSCAN_API_KEY=
PRIVATE_KEY_TESTNET=
```
